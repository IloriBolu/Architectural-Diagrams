# PRESONAL BLOG
## SCENARIO:
Your friend wants a personal blog. The site is just HTML, CSS, and images. No server-side code. They want a custom domain name and fast loading worldwide.

## GOAL:
Design an architecture that hosts a static website with a custom domain and global speed.

### DIAGRAM INCLUDES:
-	Where the website files are stored
-	How users reach the site using a custom domain
-	How the site loads fast for users in different countries
-	How HTTPS (SSL) is set up
