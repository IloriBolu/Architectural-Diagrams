# IMAGE THUMBNAIL SERVICE
## SCENARIO:
A photo app needs to automatically create thumbnail versions of every image a user uploads. The original and the thumbnail both need to be stored.

## GOAL:
Design an event-driven architecture that resizes images automatically after upload.

### DIAGRAM INCLUDES:
-	Where users upload the original image
-	What triggers the resize process (no polling allowed)
-	Where the thumbnail is saved
-	How the app reads both the original and the thumbnail
